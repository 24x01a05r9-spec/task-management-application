 # README – Task Management Application

## Project Title
Task Management Application

## Project Description
The Task Management Application is a full-stack web application used to create, update, delete, and track daily tasks efficiently. The application helps users organize their work using task statuses such as To Do, In Progress, and Done.

## Features
- User Authentication & Authorization
- Create New Tasks
- Update Existing Tasks
- Delete Tasks
- Track Task Status
- Responsive Design for Mobile and Web
- Dynamic Task Handling

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript
- React.js

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## Installation Steps

### 1. Clone the Repository
git clone https://github.com/your-username/task-management-app.git

### 2. Navigate to Project Folder
cd task-management-app

### 3. Install Dependencies
npm install

### 4. Start the Application
npm start
